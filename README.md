# Task 01 – Writing Better Prompts

## Overview
This task focuses on learning the fundamentals of Prompt Engineering by transforming vague prompts into clear, specific, and effective prompts. The goal is to understand how prompt design influences the quality of responses generated by Large Language Models (LLMs).

## Objective
- Identify vague or ineffective prompts.
- Rewrite them with better clarity, specificity, and context.
- Compare the outputs of the original and improved prompts.
- Analyze how prompt improvements affect AI-generated responses.

## Example

### Vague Prompt
Explain Artificial Intelligence.

### Improved Prompt
Explain Artificial Intelligence in 8 bullet points for MCA students. Include applications, advantages, disadvantages, and examples.

## Process
1. Select a vague prompt.
2. Analyze its limitations.
3. Rewrite the prompt with:
   - Clear objective
   - Target audience
   - Desired format
   - Specific requirements
4. Generate outputs for both prompts.
5. Compare and evaluate the results.

## Learning Outcomes
- Understanding Prompt Engineering basics.
- Improving communication with AI systems.
- Creating more accurate and useful AI responses.
- Learning the importance of clarity and context in prompts.

## Tools Used
- ChatGPT
- GitHub
- Prompt Engineering Techniques

## Conclusion
This task demonstrates that well-structured prompts produce more relevant, detailed, and accurate responses compared to vague prompts. Effective prompt design is a key skill for working with AI-powered tools.
